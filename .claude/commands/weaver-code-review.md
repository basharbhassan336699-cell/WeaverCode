---
description: مراجعة كود ملف أو مجلد محدد
argument-hint: [مسار الملف أو المجلد]
allowed-tools: ["Read", "Glob", "Grep"]
---

راجع الكود في: $ARGUMENTS (أو المشروع كله إن لم يُحدد)

فحص شامل يشمل:
- جودة الكود والالتزام بأفضل الممارسات
- الأداء والكفاءة
- الأمان والثغرات المحتملة
- وضوح الأسماء والتعليقات
- الأخطاء المنطقية المحتملة
- تغطية حالات الخطأ

قدّم تقريراً منظماً مع أمثلة كود مقترحة للتحسينات المهمة.

تفويض الاتصال

    
اضغط «فتح صفحة GitHub» ثم أدخل الرمز أدناه واضغط Authorize. سيتصل النظام تلقائياً — بلا توكن.

إعداد اتصال GitHub (مرة واحدة)

    

      لتفعيل زر «Authorize» بضغطة واحدة، أدخل بيانات تطبيق OAuth الخاص بك (يُحفظ على جهازك فقط).

      1) افتح github.com/settings/developers → New OAuth App.

      2) Callback URL: http://localhost:8080/oauth/callback — وفعّل Device Flow (اختياري).

      3) انسخ Client ID، وولّد Client Secret، والصقهما هنا.
